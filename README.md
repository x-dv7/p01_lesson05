Написать скрипт, генерирующий отчет о структуре файлов и папок на жестком диске.
1. Путь к анализируемой папке должен задаваться в параметре командной строки --path
2. Путь к отчету должен задаваться через параметр командной строки --report
3. Тип отчета (docx, xlsx, pdf, csv, json) должен определяться из расширения файла отчета. Если указано другое расширение, выдать сообщение об ошибке
4. Формат вывода в каждый вид отчета - на ваше усмотрение. Выводится имя, размер (для файлов, для папок - слово ПАПКА или FOLDER), дата изменения
5. Анализируется текущая папка и вложенные папки. Если встречаются ZIP-архивы, они рассматриваются как вложенные папки. Символические ссылки не анализируются
6. Не забывате про README.MD и requirements.txt
